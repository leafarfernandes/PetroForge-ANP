# PetroForge-ANP

Análise da produção de petróleo utilizando dados públicos da ANP.

## Objetivo

Desenvolver uma aplicação em Python para coletar, tratar, analisar e visualizar dados públicos de produção de petróleo e gás natural disponibilizados pela ANP.

Este projeto faz parte do ecossistema **PetroForge**, uma coleção de ferramentas voltadas à Engenharia de Petróleo e Computação Científica.
